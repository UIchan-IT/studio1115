# **App Name**: Lexical Leap

## Core Features:

- User Authentication: Secure user registration and login using email and password.
- Word List Management: Users can create, manage, and organize multiple word lists.
- Vocabulary Management: Add words and definitions to lists; import words in bulk from CSV files; mass delete words.
- Example Sentence Generation: Leverage AI to automatically generate example sentences for each word using the Genkit tool, enhancing understanding and context.
- Flashcard Learning: Spaced Repetition System (SRS) algorithm presents words needing review at optimal intervals for memory reinforcement.
- Quiz Generation: Generate random multiple-choice quizzes from the word lists to test proficiency.
- Progress Tracking Dashboard: Display the number of words learned, total words, and words needing review, along with quiz mistake frequency per word.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey knowledge and focus.
- Background color: Very light gray (#F0F0F0) for a clean, distraction-free learning environment.
- Accent color: Teal (#008080) for highlights and call-to-action buttons, complementing the primary color.
- Headline font: 'Space Grotesk' sans-serif for a computerized feel; body font: 'Inter' sans-serif for longer text.
- Use clean, minimalist icons for navigation and word categories.
- Maintain a clear, uncluttered layout with ample spacing for readability.
- Incorporate subtle animations for feedback during quizzes and SRS reviews.
